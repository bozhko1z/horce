Добавяне на концерти и обяви

Тази страница записва директно във вашето GitHub хранилище — без да е необходимо да пишете код. Достъпна е само за хора, които познават адреса на тази страница и имат GitHub токен; не е свързана към публичното меню на сайта.

1. Връзка с GitHub хранилището

Направете това веднъж на това устройство. Данните се пазят само в този браузър.

Създайте „Fine-grained token“ от GitHub → Settings → Developer settings, само за това хранилище, с право „Contents: Read and write“. Не споделяйте този токен.

Нов концерт

Попълнете след всяко изпълнение. Снимките се качват автоматично в правилната папка.

Можете да изберете няколко снимки наведнъж. По-малки файлове (под ~5 МБ) се качват по-бързо.

Съществуващи концерти

Свържете се с GitHub по-горе, за да видите списъка.

Нова обява за събитие

Попълнете преди всяко голямо предстоящо събитие, за да го видят посетителите на сайта.

Свържете се с GitHub по-горе, за да видите списъка.
